# AI Customer Support - Multi-Agent Coordination System

## Project Overview

This project is an AI-powered customer support system built using multiple specialized AI agents.

Current implementation includes:

- Coordinator Agent
- Planner Agent
- Intent Agent
- Knowledge Agent

The project is being developed according to enterprise Multi-Agent Architecture principles.

---

## Tech Stack

- Python
- Google Gemini API
- LangChain
- JSON
- Git & GitHub

## Run Project

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```
GOOGLE_API_KEY=YOUR_API_KEY
```

Run

```bash
python app.py
```
